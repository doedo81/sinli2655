# ⚔️ 성장 시스템 기획서 (v0.1 초안)

> 아침칠판 앱(sinli2655)에 붙이는 **학생 개인 성장·상벌점 게임 시스템**.
> 클래스123 + 상태창의 장점을 합쳐, 학생이 게임처럼 접근하도록 만든다.
> **결정 완료:** 교과 중심 능력치 / 경고 게이지 별도(레벨 안 깎음) / 상세 기획 먼저.
> 이 문서의 숫자·이름은 모두 **초안**이며 선생님이 자유롭게 고친다.

---

## 1. 목표

- 상점 = 성장의 재미(경험치·레벨·능력치), 벌점 = 별도의 부드러운 경고
- 아침칠판과 **같은 앱·같은 명단·같은 로그인**을 쓴다 (별도 앱 아님)
- 무료·무제한 (상태창의 "100레벨 제한" 같은 벽 없음)

---

## 2. 핵심 개념 한눈에

| 요소 | 설명 | 학생이 느끼는 것 |
|---|---|---|
| **경험치(XP)** | 상점을 받으면 XP 획득 | "포인트가 쌓인다" |
| **레벨** | XP가 차면 레벨업 | "내가 성장한다" |
| **능력치(스탯)** | 상점 줄 때 교과를 골라 해당 능력 +1 | "나는 수학이 세다" |
| **직업(전직)** | 특정 레벨에서 직업 선택 | "나는 수학자다" |
| **미션(퀘스트)** | 선생님이 등록 → 완료 시 보상 | "목표가 있다" |
| **경고 게이지** | 벌점은 레벨과 분리된 경고로 쌓임 | "조심해야겠다" (좌절 X) |

## 3. 능력치 (RPG 스타일 이름 + 교과 매핑) ✅ 확정

이름은 RPG 클래식 스탯, 속은 교과·활동에 연결. 상점 줄 때 능력치 +1, 동시에 XP 획득.

### 능력치 6종

| 능력치 | 뜻 | 어떤 활동으로 오르나 |
|---|---|---|
| 🧠 **지력** | INT | 수학·과학, 단원평가 |
| 📖 **지혜** | WIS | 국어·영어·독서, 주제일기 |
| 💪 **체력** | STR | 체육·활동, 밥·우유 |
| 🏹 **민첩** | DEX | 미술·실과·손재주 활동 |
| 👑 **매력** | CHA | 발표·리더십·1인1역 |
| 🛡️ **정신** | 인성 | 도덕·배려·성실, 수업준비·숙제 |

### 규칙
- 능력치는 **줄어들지 않는다** (성장만 함)
- 능력치 총합/최고치로 직업 추천에 활용
- 학생 상태창에 막대/숫자로 표시

---

## 3-2. 상점 편성표 (어떤 활동에 무슨 상점) ✅ 선생님 확정

상점 하나마다 **①능력치 ②XP ③주는 방식**을 정한다. **자주 주는 습관은 작게, 특별 성취는 크게** → 밸런스 유지.

| 상점 항목 | 오르는 능력치 | XP | 방식 | 빈도 |
|---|---|---|---|---|
| 수업준비 완료 | 🛡️ 정신 | 5 | 원터치 버튼 | 매 교시 |
| 1인1역 완료 | 👑 매력 | 5 | 원터치 버튼 | 매일 |
| 밥 다 먹기 | 💪 체력 | 5 | 원터치 버튼 | 매일 |
| 우유 다 먹기 | 💪 체력 | 5 | 원터치 버튼 | 매일 |
| 발표하기 | 👑 매력 | 10 | 원터치 버튼 | 수시 |
| 숙제 해오기 | 🛡️ 정신 | 10 | 원터치 버튼 | 숙제 있는 날 |
| 주제일기 제출 | 📖 지혜 | 20 | 반복 미션(주1회) | 주 1회 |
| 주제일기 최고득점 | 📖 지혜 | 30 | 이벤트(1명) | 주 1회 |
| 단원평가 향상점수 | 🧠 지력(교과선택) | 20 | 이벤트 | 단원평가 때 |
| 단원평가 만점 | 🧠 지력(교과선택) | 30 | 이벤트 | 단원평가 때 |

**편성 원리**
- 매일 습관(수업준비·밥·우유·1인1역) = 5점 (꾸준함 보상)
- 노력·성취(발표·숙제·주제일기) = 10~20점
- 특별 성취(만점·향상·최고득점) = 30점 (가끔이라 밸런스 OK)
- **"만점"과 "향상점수"를 함께** → 잘하는 학생·부족한 학생 **모두 성장** 가능
- UI: 학생 카드 누르면 [습관 / 노력·성취 / 특별] 3묶음 버튼 → 원터치로 XP·능력치 자동 반영
- 단원평가·발표는 교과에 따라 능력치가 달라질 수 있어 **버튼 누를 때 교과 선택** 가능하게

> 조정 가능: 하루 상점 횟수가 많으면 습관 XP를 낮추고, 적으면 올린다.

## 4. 레벨 곡선 ✅ 구현됨 (조정 가능)

- 레벨업 필요 XP: `필요XP = 50 + (현재레벨 - 1) × 30` (초반 빠르게, 뒤로 갈수록 천천히)
- 코드 상수 `RPG_XP_BASE=50`, `RPG_XP_STEP=30` 만 바꾸면 속도 조절

| 구간 | 그 레벨업에 필요한 XP | 비고 |
|---|---|---|
| Lv 1 → 2 | 50 XP | 첫날 바로 레벨업(습관 몇 번) |
| Lv 2 → 3 | 80 XP | |
| Lv 5 → 6 | 170 XP | |
| Lv 9 → 10 | 290 XP | **Lv10 전직!** |
| Lv 1 → 10 누적 | 약 1,530 XP | 활발히 쓰면 약 한 달 |

> **레벨 상한 없음.** 실제 써보고 너무 느리면 BASE/STEP를 낮춰 빠르게 조정.

## 5. 직업·전직 시스템 (클래식 RPG) ✅ 확정

| 시점 | 단계 | 내용 |
|---|---|---|
| Lv 1~9 | 견습생 | 아직 직업 없음 |
| **Lv 10** | 전직 | "전직!" 연출 + 직업 선택(자유 선택) |
| Lv 20 | 2차 전직(선택) | 상위 직업으로 승급 (나중에 추가) |

### 직업 목록 (판타지 클래스)

| 직업 | 어울리는 능력치 | 특성(보너스) |
|---|---|---|
| ⚔️ 기사 | 체력·정신 | 리더/보호. 체력 상점 XP +20% |
| 🔮 마법사 | 지력 | 지력 상점 XP +20% |
| ✨ 성직자 | 정신·매력 | 친구 돕기(선행) 보상 2배 |
| 🏹 궁수 | 민첩 | 민첩 상점 XP +20% |
| 🎵 음유시인 | 매력·지혜 | 발표·주제일기 XP +20% |

> 직업은 언제든 추가 가능(도적·현자·연금술사 등). 직업은 **칭호 + 작은 보너스** 중심(재미·정체성).

## 6. 미션(퀘스트) 시스템 (초안)

선생님이 미션을 등록 → 학생이 완료하면 선생님이 체크 → 보상 지급.

- **개인 미션**: "독서록 3편 쓰기"
- **전체 미션**: "우리 반 일주일 지각 0회"
- **반복 미션**: "매일 아침 인사" (매일/매주 초기화)

보상: XP · 특정 능력치 · 칭호 중 선택. (예: XP +30, 언어력 +2)

---

## 7. 경고 게이지 (벌점) — 레벨과 분리 ✅ 확정

- 벌점을 주면 **경고 게이지**만 오른다 (0~100). **레벨·XP·능력치는 안 깎음.**
- 벌점 1회 = **+20**
- **60점** 넘으면 선생님 화면에 알림 표시
- **선행으로 회복 −10**: 쓰레기 줍기, 친구 1인1역 돕기 등
- **매주 월요일 자동 −20** (새 마음으로 시작)
- ✨성직자 직업이면 친구의 경고까지 낮춰주는 "치유" 컨셉 가능
- 학생에겐 "빨간 막대가 차오르니 조심하자" 정도의 부드러운 신호

> 목적: 하위권 학생이 좌절하지 않게. 성장은 성장대로, 경고는 경고대로.

## 8. 화면 구성 (아침칠판에 추가할 "⚔️ 성장" 탭)

1. **학급 전체 보기**: 학생 카드 그리드(레벨·직업·상/벌점 버튼) ← 선생님이 빠르게 상벌점 주는 화면
2. **개인 상태창**: 한 명 클릭 → 레벨·경험치바·능력치·직업·미션·경고게이지
3. **미션 관리**: 미션 등록/수정/완료 체크
4. **설정**: 능력치 이름, 직업 목록, 레벨 곡선, 상점당 XP 등
5. **학생용 보기 모드**: TV/태블릿에 상태창만 크게 (`?mode=view` 방식)

## 9. 데이터 구조 (Firebase, 기존 것 안 건드림)

```
rooms/신리/
  ├─ (기존: groupScores, seating, calendarEvents ...)  ← 그대로 둠
  └─ rpg/
       ├─ config/
       │    ├─ stats        # 능력치 정의 [언어력, 논리력, ...]
       │    ├─ jobs         # 직업 정의
       │    ├─ levelCurve   # 레벨 곡선 설정
       │    └─ xpPerPoint   # 상점당 XP
       ├─ students/
       │    └─ <학생번호>/
       │         name, level, xp,
       │         stats: { 언어력, 논리력, 탐구력, 예술성, 활력, 인성 },
       │         job, title, warning,
       │         missions: { <미션id>: 완료여부 }
       ├─ missions/
       │    └─ <미션id>/ { title, reward, type }
       └─ log/            # (선택) 상벌점 기록: 누가·언제·무엇을
```

- 학생 명단은 **자리배치용 명단을 재활용** (새로 안 만듦)
